### Запуск бота

1. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv env
source env/bin/activate # Для Linux/Mac
env\Scripts\activate # Для Windows
```

2. Установите зависимости:
```
pip install -r requirements.txt
```

3. Запуск бота
```bash
python bot.py
```
